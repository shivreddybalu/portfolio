✅ SETUP IN 2 MINUTES:

1️⃣  GET FREE API KEY
   → Visit: https://makersuite.google.com/app/apikey
   → Sign in with Google
   → Click "Create API Key"
   → Copy the key (starts with: AIzaSy...)

2️⃣  ADD KEY TO PROJECT
   → Open: src/environments/environment.ts
   → Replace: 'YOUR_GEMINI_API_KEY_HERE'
   → With: 'AIzaSy...' (your actual key)

3️⃣  ALSO UPDATE PRODUCTION
   → Open: src/environments/environment.prod.ts
   → Add the same key

4️⃣  TEST IT!
   → Run: ng serve
   → Open chatbot
   → Look for "LIVE" badge = Real AI is active!
   → Look for "Quick Responses" = Using fallback mode

🔒 SECURITY TIP:

For production, restrict your API key:
1. Go to: https://console.cloud.google.com/
2. Find your API key
3. Add HTTP referrer restriction
4. Enter: https://yourdomain.com/*

This prevents others from using your key!
